## Проектная работа "Mesto-react"
#### Проектная работа представляет собой портированый на React проект Mesto.

Для развертывания проекта необходимо:
- Запустить "GitBash"("Terminal" на Mac OS);
- Выполнить команду "git clone https://github.com/Gonzoooo/mesto-react.git";
- Выполнить команду "cd mesto";
- Выполнить команду "npm i webpack --save-dev";
- Выполнить команду "npm run build";
- Выполнить команду "npm run dev".
